# mineSweeper
ブラウザ(chrome等)で動くマインスイーパー

## 作成した目的
コーディングの練習用に作成  
なので見た目は...

## 使い方
・「マインスイーパーを作成」からマス数と地雷数を変更可能  
・左クリックでマスを開く  
・右クリックでフラグを建てます

## ※注意※
マス数が多いと・・・  
・スタックオーバーフローが発生！  
・マスが潰れて見た目が変に！？  
等の問題が発生します
